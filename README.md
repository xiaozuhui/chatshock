## 主要功能

### 文件模块

- [X]  主动接入minio
- [ ]  包装、隐藏 url（以及刷新url）和文件名、bucket之间的关系
- [ ]  提供统一的接口和实现，以应对不同文件服务器的功能

### 用户模块

- [X]  手机号注册

### 聊天室

- [ ]  websocket
